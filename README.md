# Blinkit-Analysis-Python-project-

## Problem Statement 
Analysis of Blinkit's sales performance, customer satisfaction, and inventory distribution to identify key insights and KPI's Requirements such as Total Sales, Average sales, number of items sold.

## Tools Used
Python (Pandas,Numpy,Matplotlib,Seaborn libraries)

## Key insights
- Analyzed the impact of fat content on total sales and ientified that low-fat items account for 65% of overall sales, indicating strong customer preference for healthier product options.
- Fruits and vegetables have highest sales among all items types
